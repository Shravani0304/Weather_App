# 🌦️ Weather App

## 📌 Description

This is a simple Weather Application that shows real-time weather information of any city.  
Users can search for a city and get details like temperature, humidity, and weather conditions.

---

## 🚀 Features

- Search weather by city name
- Displays temperature
- Shows humidity and weather condition
- User-friendly interface
- Real-time data using API

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Weather API (like OpenWeatherMap)
